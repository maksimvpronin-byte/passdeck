# Проверка сборки PassDeck 0.2.0

Проверено на Windows с Node.js 22.22.0:

- `npm run lint` — успешно;
- `npm run typecheck` — успешно;
- `npm test` — 3 test files, 9 tests успешно;
- production build renderer — успешно;
- Electron Windows portable package — успешно;
- итоговый файл создаётся один раз;
- `release/win-unpacked` удаляется после упаковки;
- `apps/desktop/release` не используется.

Проверенные пользовательские сценарии:

- создание, открытие, изменение и сохранение KDBX;
- повторное открытие сохранённой базы;
- обычные и защищённые пользовательские поля;
- показ и изменение существующего пароля;
- добавление, экспорт, повторное открытие и удаление вложений;
- запрет вложений с совпадающим именем;
- лимит 25 МБ на файл и 100 МБ на запись;
- drag-and-drop записи между группами;
- сохранение новой группы записи после повторного открытия;
- Auto-Type по `Ctrl+Alt+A`;
- фиксированная последовательность `{USERNAME}{TAB}{PASSWORD}{ENTER}`;
- отсутствие принудительного восстановления, активации или изменения размера целевого окна;
- отмена Auto-Type при смене активного окна.

Ожидаемый единственный артефакт:

```text
release\PassDeck-Portable-0.2.0-x64.exe
```
