# Changelog

All notable changes to PassDeck will be documented in this file.

## [0.3.0-touchid] - 2026-06-27

### 🔓 Добавлено: Биометрическая разблокировка (Touch ID / Face ID) на macOS

**Ключевые возможности:**
- Разблокировка базы данных через Touch ID или Face ID на macOS устройствах
- Интеграция с системными биометрическими диалогами macOS
- Безопасное шифрование токенов AES-256-GCM с HMAC-SHA256 аутентификацией
- Хранение сессий только в защищённом Keychain macOS
- Fallback в мастер-пароль при любой ошибке биометрической авторизации
- Короткоживущие токены (максимум 5 минут времени жизни)

**Архитектура:**
- Новый `BioAuthService` в main-процессе для обработки биометрии
- Два новых IPC канала: `auth:bio-init` и `auth:bio-unlock`
- UI компонент кнопки Touch ID в модалках разблокировки
- Модалка первой настройки биометрической авторизации

**Безопасность:**
- Все секреты хранятся только через macOS Keychain (не в памяти)
- Не передаём биометрические данные в логи или renderer процесс
- Фоллбек в мастер-пароль при любых ошибках биометрии
- Whitelisted IPC каналы с строгой типизацией

**Файлы:**
- `apps/desktop/src/main/services/bio-auth-service.ts` — сервис авторизации
- `apps/desktop/src/main/types/bio-token.ts` — типы для IPC
- `apps/desktop/src/main/services/bio-auth-context.ts` — контекст для IPC
- `apps/desktop/src/renderer/auth.ts` — кнопка Touch ID (BioUnlockButton)
- `apps/desktop/src/renderer/components/BioSetupModal.tsx` — модалка настройки
- `apps/desktop/src/renderer/styles/bio-auth.css` — стили биометрии

### ⚡ Улучшения

- Интеграция биометрии в существующую архитектуру Auto-Type и DatabaseService
- Поддержка нескольких зарегистрированных биометрических данных (Touch ID / Face ID)

## [0.2.1] - 2026-06-26

### 🖥️ Добавлено: Windows Auto-Type через Ctrl+Alt+A

- Глобальная горячая клавиша для автозаполнения логина/пароля
- Фиксированная последовательность `{USERNAME}{TAB}{PASSWORD}{ENTER}`
- Автозаполнение без активного окна PassDeck и принудительное восстановление
- Проверка фокуса при запуске (отмена если окно изменилось)

### 📋 Добавлено: Пользовательские поля KDBX

- Поддержка обычных и защищённых пользовательских полей
- В редакторе: показ только незащищённых значений
- Явное раскрытие защищённого поля для копирования
- Сохранение скрытых значений без передачи в renderer

### 📎 Добавлено: Вложения KDBX

- Добавление нескольких файлов к записи (до 25 МБ на файл, до 100 МБ суммарно)
- Просмотр имени и размера файла
- Экспорт через системный диалог
- Удаление после подтверждения
- Байты вложений остаются в main-процессе

### 🎯 Добавлено: Drag-and-drop перенос записей между группами

- Захват записи из центральной таблицы
- Перетаскивание на целевую группу в дереве
- Подсветка группы назначения
- Удаление кнопок и модалок перемещения

## [0.2.0] - 2026-06-25

Первый релиз функционально завершённого приложения с поддержкой KDBX 4.x, нескольких баз в окнах, автоблокировкой, генератором паролей и другими функциями.
