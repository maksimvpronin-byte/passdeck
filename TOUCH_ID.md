# PassDeck Touch ID — Биометрическая разблокировка на macOS

## Краткое описание

PassDeck 0.3.0 поддерживает разблокировку базы данных через Touch ID или Face ID на устройствах macOS 13 (Ventura) и новее.

## Требования

- **macOS:** 13 (Ventura) или новее
- **Hardware:** Mac с поддержкой Touch ID в логин-кнопке или Face ID
- **Ключевой цепи (Keychain):** Конфигурируется автоматически

## Установка

### На первый запуск на новом Mac

При первом запуске PassDeck на macOS будет предложена модалка настройки биометрии:

1. Нажмите кнопку **"Регистрировать биометрию"**
2. Откроется системный диалог macOS для регистрации Touch ID / Face ID
3. Задержите отпечаток пальца или сфотографируйте лицо
4. Дождитесь подтверждения успешной регистрации

### Разблокировка базы

1. Откройте PassDeck
2. Если база заблокирована, появится кнопка **"Разблокировать по Touch ID"**
3. Нажмите кнопку
4. Откроется системный диалог Touch ID / Face ID
5. Успешная разблокировка → мастер-пароль из памяти очищается

### Безопасность

**🔒 macOS Keychain:**
- Все секреты хранятся только в защищённом Keychain macOS
- Короткоживущие токены (< 5 минут)
- Биометрические данные не передаются за пределы системы

**🛡️ Master Password Fallback:**
При любой ошибке биометрии:
- Возвращается требование ввести мастер-пароль
- Это безопасно — мастер-пароль является единственным способом доступа

**⚠️ Важно:**
Биометрические данные хранятся локально и не передаются никуда. Apple отвечает за безопасность Keychain.

## Ограничения

- Работает только на macOS (Windows/Linux биометрия отдельная задача)
- Требует наличия зарегистрированного Touch ID / Face ID в настройках безопасности системы
- Для разблокировки базы нужно, чтобы биометрическое устройство было зарегистрировано в системе

## Технические детали

- **Шифрование:** AES-256-GCM с HMAC-SHA256 аутентификацией
- **Key derivation:** PBKDF2 для вывода ключей из мастер-пароля
- **Session timeout:** 5 минут (токены живут < 1 минуты если кэшируются)

## Отключение биометрии

Чтобы временно отключить разблокировку по Touch ID:
- Используйте мастер-пароль вместо кнопки
- Зайдите в настройки macOS и отключите Touch ID в настройках безопасности

## Решение проблем

### "Кнопка Touch ID не появляется"

1. Убедитесь, что macOS 13 или новее
2. Проверьте настройки: Системы → Touch ID/Face ID → разрешено для приложений
3. Перезапустите PassDeck после настройки биометрии в системе

### "Разблокировка не работает"

1. Попробуйте ввести мастер-пароль (fallback)
2. Перезарегистрируйте Touch ID в настройках macOS
3. Проверьте, что приложение имеет доступ к Keychain
