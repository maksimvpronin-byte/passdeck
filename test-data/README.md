# PassDeck Demo

Файл `PassDeck-Demo.kdbx` содержит только вымышленные данные.

Мастер-пароль:

```text
PassDeck-Demo-2026!
```

Формат: KDBX 4.1, AES-256, Argon2id.

Содержимое:

- древовидные группы;
- три тестовые записи;
- теги;
- пользовательские поля;
- текстовое вложение `demo.txt`.

Не используйте этот пароль и данные в реальных базах.
